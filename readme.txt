== Description ==

Simple Admin plugin that lets you choose the font style and size for the Edit Posts / Pages test editor. Works only with HTML editor and the reply field of Comments in WordPress admin. Also works with the Fullscreen / Distraction Free Editor. Comes with a preview feature.

More info is available <a href="http://www.geniusstartup.com/wordpress-admin-font-editor-plugin/">here</a>.

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > Admin Font Editor to adjust the font size and font style

== Frequently Asked Questions ==

= Does this plugin work with both the Visual and Text editor?  =

No, only the Text editor.

= Will the new font style be visible to all users? =

No, just the logged in user. That way, every user can choose the font and font size they prefer.

= Does it change anything on the public pages / front-end of my site? =

No.

= How do I go back to the WordPress default styling? =

Just deactivate the plugin.
